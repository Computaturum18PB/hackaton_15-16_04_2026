# Заметки - справочник функций библиотеки SymPy

## Создание символов и выражений

- `symbols('x y z')` - создание одного или нескольких символьных переменных;
- `Symbol('x')` - создание одного символа;
- `sympify('x**2 + 2*x + 1')` - преобразование строки в символьное выражение;
- `S(1/2)` - создание точного рационального числа (1/2 вместо 0.5);

## Основные математические операции

- `simplify(expr)` - упрощение выражения;
- `expand(expr)` - раскрытие скобок;
- `factor(expr)` - разложение на множители;
- `collect(expr, x)` - группировка членов по степеням x;
- `cancel(expr)` - сокращение дробей;
- `apart(expr)` - разложение на простые дроби;
- `together(expr)` - приведение к общему знаменателю;

## Производные и интегралы

- `diff(expr, x)` - первая производная по x;
- `diff(expr, x, 2)` - вторая производная по x;
- `diff(expr, x, y)` - смешанная производная (сначала по x, затем по y);
- `Derivative(expr, x)` - невычисленная производная (символьное представление);
- `integrate(expr, x)` - неопределенный интеграл по x;
- `integrate(expr, (x, a, b))` - определенный интеграл от a до b;
- `Integral(expr, x)` - невычисленный интеграл;

## Пределы

- `limit(expr, x, 0)` - предел при x → 0;
- `limit(expr, x, oo)` - предел при x → ∞ (oo - бесконечность);
- `limit(expr, x, 0, dir='-')` - предел слева;
- `limit(expr, x, 0, dir='+')` - предел справа;
- `Limit(expr, x, 0)` - невычисленный предел;

## Решение уравнений

- `solve(expr, x)` - решение уравнения expr = 0 относительно x;
- `solve([expr1, expr2], [x, y])` - решение системы уравнений;
- `solveset(expr, x, domain=S.Reals)` - решение с указанием множества (действительные числа);
- `linsolve([eq1, eq2], [x, y])` - решение системы линейных уравнений;
- `nonlinsolve([eq1, eq2], [x, y])` - решение системы нелинейных уравнений;
- `dsolve(eq, f(x))` - решение дифференциального уравнения;

## Подстановка и вычисление

- `expr.subs(x, 2)` - подстановка значения вместо символа;
- `expr.subs([(x, 2), (y, 3)])` - множественная подстановка;
- `expr.subs(x, y)` - замена символа x на y;
- `expr.evalf()` - численное вычисление с плавающей точкой;
- `expr.evalf(subs={x: 2}, n=10)` - вычисление с 10 знаками точности;
- `N(expr, 15)` - вычисление с 15 знаками точности (альтернатива evalf);

## Тригонометрические функции

- `sin(x), cos(x), tan(x)` - тригонометрические функции;
- `asin(x), acos(x), atan(x)` - обратные тригонометрические;
- `trigsimp(expr)` - упрощение тригонометрических выражений;
- `expand_trig(expr)` - раскрытие тригонометрических функций;
- `rewrite(expr, cos)` - переписать через косинусы;

## Экспоненты и логарифмы

- `exp(x)` - экспонента e^x;
- `log(x)` - натуральный логарифм ln(x);
- `log(x, b)` - логарифм по основанию b;
- `logcombine(expr)` - объединение логарифмов;
- `powsimp(expr)` - упрощение степеней;
- `expand_log(expr)` - раскрытие логарифмов;

## Матрицы и линейная алгебра

- `Matrix([[1, 2], [3, 4]])` - создание матрицы;
- `Matrix(3, 3, lambda i, j: i+j)` - создание с помощью функции;
- `M.det()` - определитель матрицы;
- `M.inv()` - обратная матрица;
- `M.T` - транспонирование;
- `M.eigenvals()` - собственные значения;
- `M.eigenvects()` - собственные векторы;
- `eye(3)` - единичная матрица 3x3;
- `zeros(2, 3)` - нулевая матрица;
- `ones(3, 3)` - матрица из единиц;

## Векторы и геометрия

- `Point(0, 0)` - точка на плоскости;
- `Line(p1, p2)` - прямая через две точки;
- `Circle(center, radius)` - окружность;
- `Triangle(p1, p2, p3)` - треугольник;
- `Polygon(*points)` - многоугольник;

## Работа с множествами

- `Interval(0, 5)` - интервал [0, 5];
- `Interval(0, 5, left_open=True, right_open=True)` - интервал (0, 5);
- `FiniteSet(1, 2, 3)` - конечное множество;
- `Union(A, B)` - объединение множеств;
- `Intersection(A, B)` - пересечение;
- `Complement(A, B)` - разность множеств;

## Печать и отображение

- `init_printing()` - включение красивого вывода (LaTeX в Jupyter);
- `pprint(expr)` - красивый вывод в консоль;
- `latex(expr)` - генерация LaTeX кода;
- `print(latex(expr))` - вывод LaTeX строки;

## Константы и специальные символы

- `pi` - число π;
- `E` - число e (экспонента);
- `I` - мнимая единица (√-1);
- `oo` - бесконечность;
- `nan` - не число (Not a Number);
- `EulerGamma` - постоянная Эйлера-Маскерони;
- `Catalan` - постоянная Каталана;

## Дифференциальные уравнения

- `Eq(f(x).diff(x, x) + f(x), 0)` - создание ДУ (f'' + f = 0);
- `dsolve(eq, f(x))` - решение ДУ;
- `dsolve(eq, f(x), ics={f(0): 1, f(x).diff(x).subs(x,0): 0})` - решение с начальными условиями;
- `laplace_transform(f(t), t, s)` - преобразование Лапласа;
- `inverse_laplace_transform(F(s), s, t)` - обратное преобразование Лапласа;

## Серии и разложения

- `series(expr, x, 0, 6)` - разложение в ряд Тейлора (до 6 степени);
- `expr.series(x, 0, 6)` - альтернативный синтаксис;
- `O(x**3)` - символ порядка (Big O);
- `expr.removeO()` - удаление O() из ряда;

## Преобразования

- `apart(expr, x)` - разложение на простые дроби;
- `together(expr)` - приведение к общему знаменателю;
- `ratsimp(expr)` - упрощение рациональных выражений;
- `radsimp(expr)` - упрощение с корнями;
- `fraction(expr)` - числитель и знаменатель (возвращает кортеж);

## Логические выражения

- `And(A, B)` - логическое И;
- `Or(A, B)` - логическое ИЛИ;
- `Not(A)` - логическое НЕ;
- `Xor(A, B)` - исключающее ИЛИ;
- `Implies(A, B)` - импликация;
- `satisfiable(expr)` - проверка выполнимости логического выражения;
- `simplify_logic(expr)` - упрощение логического выражения;

## Суммы и произведения

- `Sum(expr, (i, a, b))` - сумма от i=a до b;
- `Product(expr, (i, a, b))` - произведение от i=a до b;
- `summation(expr, (i, a, b))` - вычисление суммы;
- `product(expr, (i, a, b))` - вычисление произведения;

## Физика и специальные функции

- `gamma(x)` - гамма-функция;
- `beta(x, y)` - бета-функция;
- `hyper([a, b], [c], z)` - гипергеометрическая функция;
- `airyai(x)` - функция Эйри;
- `besselj(nu, x)` - функция Бесселя первого рода;
- `dirac(x)` - дельта-функция Дирака;
- `Heaviside(x)` - функция Хевисайда;
