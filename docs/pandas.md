# Заметки - справочник функций библиотеки Pandas

## Создание структур данных

- `pd.Series()` - создание одномерного массива;
- `pd.DataFrame({Данные}, index=[Наименования])` - создание двумерного массива;
- `pd.read_csv()` - загрузка данных из CSV файла;
- `pd.read_excel()` - загрузка данных из Excel файла;

## Срезы данных 

- `.head()` - первые n строк;
- `.tail()` - последние n строк;

## Атрибуты таблиц

- `.shape` - размерность таблицы;   
- `.index` - индексы строк;  
- `.columns` - список названий столбцов;
- `.describe()` - статистика по числовым колонкам; 
- `.values` - значения в виде numpy массива;
- `.size` - общее количество элементов;
- `.empty` - проверка на пустоту (True/False);

## Выбор и фильтрация данных

- `df['колонка']` - выбор одной колонки (возвращает Series);  
- `df[df['колонка'] > значение]` - фильтрация по условию;  
- `.isin()` - фильтрация по вхождению в список;  
- `.query()` - фильтрация с помощью строкового выражения;  

## Работа с пропусками

- `.isnull()` - проверка на пропуски (True/False);
- `.dropna()` - удаление строк с пропусками;
- `.fillna(значение)` - заполнение пропусков указанным значением;

## Преобразование данных

- `.rename(columns={'старое':'новое'})` - переименование колонок;  
- `.drop(колонки, axis=0/1)` - удаление строки/столбца;  
- `.drop_duplicates()` - удаление повторяющихся строк;  
- `.sort_values('колонка')` - сортировка по колонке;  
- `.set_index('колонка')` - установка колонки как индекс;  

## Агрегация и группировка

- `.sum(), .mean(), .std()` - статистика по колонкам ([пример](./numpy.md#методы-массивов));  
- `.min(), .max(), .count()` - минимум, максимум, количество;  
- `.value_counts()` - подсчет уникальных значений;  

## Объединение данных

- `pd.concat([df1, df2])` - склеивание по строкам/столбцам;
- `.join(df2)` - объединение по индексам;

## Применение функций

- `.apply(функция)` - применение функции к каждой строке/колонке;
- `.applymap(функция)` - применение к каждому элементу (DataFrame);
- `.map(словарь)` - замена значений по словарю (Series);
- `.replace({старое, новое})` - замена значений из словаря;

## Работа с временем

- `pd.to_datetime()` - преобразование в datetime тип;  
- `.dt.year/month/day` - извлечение части даты;

## Вывод

- `.to_csv()` - сохранение в CSV;  
- `.to_excel()` - сохранение в Excel;  
- `.to_numpy()` - преобразование в numpy массив;
