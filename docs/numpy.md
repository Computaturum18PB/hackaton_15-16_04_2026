# Заметки - справочник функций библиотеки Numpy

- `.array()` - создание массива numpy;
- `.zeros()` - создание массива из нолей;
- `.ones()` - создание массива из единиц;
- `.arange()` - создание массива с фиксированным шагом;
- `.linspace()` - создание равномерно-распределенного массива;
- `.full()` - создание массива из элемента;
- `.eye()` - создание единичной матрицы
---
- `.ndim` - измерение массива;
- `.shape` - размерность массива
