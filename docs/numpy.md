# Заметки - справочник функций библиотеки Numpy

## Функции создания массивов

`np.array()` - создание массива;  
`np.zeros()` - создание массива из нолей;  
`np.ones()` - создание массива из единиц;  
`np.arange()` - создание массива с фиксированным шагом;  
`np.linspace()` - создание равномерно-распределенного массива;  
`np.full()` - создание массива из одного элемента;  
`np.eye()` - создание единичной матрицы;  

## Функции преобразования

`np.sign()` - определение знака у элементов (-1, 0, 1);  
`np.ceil()` – округление элементов вверх;  
`np.floor()` - округление элементов вниз;  
`np.rint()` - округление до ближайшего целого числа;  
`np.where(condition, x, y)` - x если condition иначе y;  
`np.dot()` - скалярное умножение массивов;  
`np.reshape()` - изменение формы массива;  

## Методы массивов
`.mean()` - среднее арифметическое элементов;  
`.std()` - стандартное отклонение (разброс относительно среднего);  
`.sum()` - сумма всех элементов;  

## Функции ввода/вывода
`np.loadtxt(fname, delimiter, skiprows)` - загрузка данных из текстового файла;  
`np.savetxt(fname, X, delimiter)` - сохранение данных в текстовый файл;  

## Математические функции 
`np.log()` - натуральный логарифм каждого элемента;  
`np.exp()` - экспонента (e^x) каждого элемента;  
`np.clip(a(object), min, max)` - замена всех элементов меньших на min, больших на max;  

## Атрибуты массивов
`.ndim` - количество измерений массива;  
`.shape` - размерность массива;
