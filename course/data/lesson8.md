# Применение нефтепродуктов

1. Топливо и энергетика  
Основная сфера применения нефтепродуктов — производство топлива: бензин, дизельное топливо, керосин, мазут — для транспорта (автомобилей, самолётов, судов);
сжиженный газ — для отопления, в качестве энергоносителя;
котельное топливо (мазут, сланцевое масло) — для промышленных печей, тепловых установок;
реактивное топливо — для авиации;
попутный нефтяной газ (ПНГ) — как топливо для энергетических установок, сырьё для химической промышленности.


2. Пластмассы и полимеры  
Из лёгких фракций нефти получают «строительные блоки» для пластика — этилен и пропилен, которые полимеризуются с образованием различных видов пластмасс:
полиэтилен — для производства плёнок, пакетов, упаковки;
полипропилен — для изготовления жёстких деталей (крышки для бутылок, автомобильные запчасти);
нейлон, полиэстер — для текстильной промышленности (одежда, ковры, бутылки);
материалы для производства корпусов бытовой техники, гаджетов, труб, оконных рам и др.

3. Смазочные материалы и масла  
Нефтяные масла используются:
смазочные масла — для уменьшения трения и износа механизмов (двигатели, узлы машин);
антикоррозионные средства (несмазочные масла — трансформаторные, конденсаторные);
моторные, реактивные, трансмиссионные масла — в автомобильной и авиационной отраслях;
«белые» масла — в парфюмерии и медицине.

4. Синтетический каучук и резина  
Из побочных продуктов нефти создают полимеры для производства: автомобильных и велосипедных шин;
резиновых деталей для техники и оборудования;
изделий для спорта и быта (резиновые коврики, обувь).

5. Строительство и дорожные покрытия  
Нефтепродукты применяются в строительстве:
битум и гудрон — для гидроизоляции крыш, фундаментов, мостов;
асфальт (смесь битума и каменного щебня) — для дорожного покрытия;
заливочные аккумуляторные мастики;
кровельные и изоляционные материалы.

6. Бытовая химия и моющие средства   
Продукты нефтепереработки входят в состав:
стиральных порошков;
средств для мытья посуды, стёкол;
отбеливателей и пятновыводителей;
отдушек и поверхностно-активных веществ (ПАВ).

7. Косметика и парфюмерия  
Нефтепродукты используются в производстве:
кремов, мазей, лосьонов (минеральное масло, вазелин);
лаков для ногтей, теней для век, косметических карандашей (парафин, пропиленгликоль);
духов и одеколонов (искусственные ароматы на основе бензола и его производных);
средств по уходу за кожей (увлажняющие и защитные компоненты).

8. Лекарственные средства  
Из нефтепродуктов получают: аспирин, парацетамол, стрептоцид и другие медикаменты;
антимикробные и антигистаминные препараты;
антибиотики;
БАДы.

9. Синтетические ткани и текстиль  
Нефтепродукты лежат в основе производства: полиэстера, акрила, нейлона — для одежды, белья, обуви;
искусственных шуб, наполнителей для пуховиков;
функциональных тканей (термобельё, флисовые кофты).

10. Растворители и химические реагенты  
Из лёгких фракций нефти получают:
ацетон (из пропилена и бензола) — для лакокрасочной промышленности, косметики, очистки оборудования;
толуол и ксилолы — для растворения масел, смол, полимеров (краски, промывка деталей);
метанол (из синтез-газа) — как реактив, для производства формальдегида, промышленного топлива.

11. Продукты питания и корм для животных  
Нефтепродукты используются для производства:
жевательных резинок (полиэтиленовые и парафиновые смолы);
синтетических ванилина и ароматизаторов;
пищевых красителей;
искусственного белка (паприн) для корма животных.

12. Другие сферы применения  
Лакокрасочные материалы (краски, эмали, лаки);
клеи и герметики;
изоляционные материалы (для электротехники, радиоэлектроники);
компоненты для производства электроники (корпусы, детали);
сварочные электроды и кокс (в металлургии);
парафин (для свечей, спичечного производства, изоляции, смазки для лыж и сноубордов);
материалы для производства спортивных товаров (теннисные ракетки, рыболовные снасти);
компоненты для производства косметики и средств гигиены (мыло, шампуни)
